Intro To HTML

!DOCTYPE html

So first, when you are writing a webpage, you should always start with a !DOCTYPE html tag showing that the webpage was written is modern html, not those weird old types of html. It is written between smaller than and greater than signs like this (inspect and see whats there) all tags start with a less than sign and end with greater than sign

html and /html

After the !DOCTYPE html tag there will be a html tag(REMEMBER that tags ALWAYS start with a smaller than sign and end with a greater than sign) we have a pair of html tags,there is a start tag like this html and the end tag is like this /html.The start tag comes straight after the!DOCTYPE html tag and the end tag comes at the end of everything, it ends the whole html and also remember all end tags has a slash like this /html

head and /head

The head tag comes in pairs just like the html tags, it is used for the title tags and the meta tag.

meta charset="UTF-8"

The meta charset="UTF-8" tag is inside the head tag it doesn't come in pairs.It is used to specify that the webpage is in English so if you are planning the page to be in Chinese you should probably consider changing the UTF-8 into something else.

title and /title

The title tags is also inside the head tag, inside the title tags is the title of your webpage which shows the on the tabs on top of browsers for example, if you inspected the page you will see the title is HTML: HyperText Markup Language and it shows on the tab of this webpage

body and /body

The body tags contain the part of the webpage that we can see. It's end tag comes just before the html'send tag.

Now that all the essential elements needed to start the webpage have been introduced, we can talk about what goes inside the body tag, which are the tags that make up the webpage. you should have something like this after you are ready to get some information inside the page:
